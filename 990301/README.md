# Quinty (1999-03-01) Patch

- bass (ARM9 fork): https://github.com/ARM9/bass

- Put the original ROM as `./roms/quinty990301.sfc` (SHA-1: `63b4d7f471272dc5b605417bccf334740890ae59`)

### Build
- `_build.bat` - Build the patched ROM.
